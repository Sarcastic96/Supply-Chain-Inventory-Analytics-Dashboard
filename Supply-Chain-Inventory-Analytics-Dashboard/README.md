# 📦 Supply Chain & Inventory Analytics Dashboard

> An interactive Power BI dashboard for analyzing inventory performance, demand patterns, supply shortages, profitability, and operational efficiency using Business Intelligence techniques.

---

## 📖 Overview

Efficient supply chain management is essential for minimizing inventory costs, reducing shortages, and improving business profitability. This project utilizes **Power BI**, **Power Query**, **SQL**, and **DAX** to transform raw supply chain data into an interactive dashboard that provides actionable insights for inventory planning and operational decision-making.

---

## 🎯 Objectives

- Monitor inventory performance across products.
- Analyze demand, availability, and supply shortages.
- Track key business KPIs and profitability.
- Support data-driven inventory and supply chain decisions.

---

## 🛠 Tech Stack

- Microsoft Power BI
- SQL (MySQL & SQL Server)
- Power Query
- DAX
- Microsoft Excel / CSV

---

## 📂 Dataset

Large-scale supply chain dataset containing inventory, sales, demand, warehouse, supplier, and profitability information.

Key attributes include:

- Product ID
- Inventory Level
- Daily Demand
- Availability
- Supply Shortage
- Sales
- Profit
- Supplier
- Warehouse
- Product Category
